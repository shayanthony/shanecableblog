# The Code Void

This is a [Next.js](https://nextjs.org/), [Tailwind CSS](https://tailwindcss.com/) blogging starter template. This is based off of a wonderful starter template which can be found [here](https://github.com/timlrx/tailwind-nextjs-starter-blog).

The site is modified to include Spotify API integration & uses SWR hooks to update blog likes with a postgresql DB hosted on Heroku.

Check out the documentation below to give it a spin.

## Quick Start Guide

1. JS (official support)

```bash
npx degit https://github.com/shayanthony/shanecableblog.git
```

2. Personalize `siteMetadata.js` (site related information)
3. Personalize `authors/default.md` (main author)
4. Modify `projectsData.js`
5. Modify `headerNavLinks.js` to customize navigation links
6. Add blog posts
7. Deploy on Vercel

## Installation

```bash
npm install
```

## Development

First, run the development server:

```bash
npm start
```

or

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.
